# SpellRead Copyright & Content Policy

## Companion Mode (MVP)

SpellRead operates as a **reading companion**, not a book publisher.

- Users read Harry Potter using their **own physical book, Kindle, or library ebook**.
- SpellRead does **not** host, display, or distribute full chapter text.
- All learning happens through vocabulary previews, original background guides, and quizzes.

Product copy: **"Bring your own book."**

## Permitted Content

| Content Type | Policy |
|-------------|--------|
| Original background summaries | Allowed — written in our own words, no plot spoilers beyond the chapter |
| Vocabulary definitions | Allowed — standard dictionary-style definitions |
| Short example sentences | Allowed — brief illustrative phrases (≤15 words) to teach word meaning |
| Page hints | Allowed — "see p. 6" to help readers locate context in their own copy |
| Original illustrations | Allowed — Hogwarts-inspired style, no movie stills or official artwork |

## Restricted Content

| Content Type | Policy |
|-------------|--------|
| Full chapter text | **Prohibited** |
| Extended verbatim excerpts | **Avoid** — limit to single short sentences for vocabulary context |
| Movie assets / official HP artwork | **Prohibited** |
| Trademark use in product name | Use "SpellRead" brand; reference "Harry Potter" descriptively only |

## Fair Use Considerations

Brief quotations in vocabulary cards and quiz stems may qualify as fair use when:

1. The quote is **short** (typically one sentence).
2. The purpose is **educational** (vocabulary in context).
3. The quote does **not substitute** for reading the book.
4. The work is **not commercially distributed** as a replacement for the book.

**Recommendation before public launch:** Consult IP counsel for jurisdiction-specific guidance (US fair use, UK fair dealing, EU exceptions).

## Data & Child Safety

- COPPA-aligned: minimal PII (nickname + avatar only for children).
- Parent consent gate before account creation for users under 13.
- No public social features or leaderboards by default.
